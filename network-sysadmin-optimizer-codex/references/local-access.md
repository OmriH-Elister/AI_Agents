# Local Access Notes

Use this file for environment-specific connection details that should not live in `SKILL.md`.

Populate only the minimum durable context needed to operate safely:

- preferred remoting methods for Windows hosts
- preferred remoting methods for Linux hosts
- domain or workgroup names
- admin usernames or service account names
- host inventory conventions or critical host groups
- password manager location, secret command, or environment variable names
- maintenance-window expectations

Do not store plaintext passwords or long-lived tokens here.

If you are migrating details from `<REDACTED>`, copy only non-secret operating context and move secrets into a secure store.
